# 🎮 Pixel Dungeon Action

HTML5 Canvas 기반의 아케이드 스타일 픽셀 던전 액션 게임 프로토타입입니다. 
인터넷 브라우저만 있으면 설치 없이 링크 클릭 한 번으로 즉시 게임을 즐기고 테스트할 수 있습니다.

특히 **치지직(Chzzk) 실시간 방송 채팅 연동 기능**이 내장되어 있어, 실제 스트리머의 라이브 방송 채팅을 게임 속 적과 아군의 말풍선으로 구현할 수 있습니다.

👉 **[지금 바로 웹에서 플레이하기](https://unrankedguest.github.io/pixel-dungeon/)**

---

## 💬 치지직 실시간 채팅 연동 참여 방법

게임 우측 상단의 **톱니바퀴 버튼(⚙️)**을 누르면 채팅 연동 설정 패널이 열립니다.

1. **중계 서버 연결:** 중계 서버 주소(Socket.io) 칸에 구동 중인 중계 서버 주소(예: `https://chzzk-chat-proxy.onrender.com` 또는 로컬 서버 주소)를 기입합니다.
2. **방송 주소 입력:** 스트리머의 채널 전체 URL 주소(예: `https://chzzk.naver.com/live/...`)를 `치지직 채널 ID 또는 방송 URL` 입력창에 붙여넣습니다.
3. **연동 시작:** **[연동 시작]** 버튼을 클릭하면 실시간 채팅이 동기화되며, 게임 속 적(몬스터)들이 시청자들의 채팅을 말풍선으로 띄우며 나타납니다!

## 🕹️ 게임 조작 방법

* **`A` / `D` :** 좌우 이동
* **`Space` :** 점프 (체공 게이지가 있을 때 공중에서 한 번 더 누르고 있으면 제트팩 체공 작동)
* **`마우스 조준 & 클릭` :** 탄환 발사
* **`⚙️ (톱니바퀴)` :** 치지직 연동 패널 열기/닫기
