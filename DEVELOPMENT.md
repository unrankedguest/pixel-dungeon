# 🎮 Pixel Dungeon Action Prototype

HTML5 Canvas 기반의 아케이드 스타일 픽셀 던전 액션 게임 프로토타입입니다. 단 하나의 HTML 파일로 구성되어 있어 별도의 복잡한 개발 환경 세팅 없이 브라우저에서 즉시 실행 및 테스트할 수 있습니다.

특히 **치지직(Chzzk) 실시간 방송 채팅 연동 기능**이 내장되어 있어, 실제 라이브 스트리밍 채팅을 게임 속 적과 아군의 말풍선으로 구현할 수 있습니다.

## 💬 치지직 실시간 채팅 연동 시스템

게임 우측 상단의 **톱니바퀴 버튼(⚙️)**을 누르면 채팅 연동 제어판이 열립니다. CORS 정책 우회를 위해 로컬 중계 서버를 띄워 접속하는 아키텍처를 사용합니다.

### 1. 로컬 중계 서버(Socket.io) 가동 방법
치지직 API는 보안 정책(CORS)으로 인해 브라우저 단에서 직접 호출이 불가능합니다. 이를 우회하기 위해 Node.js 환경에서 돌아가는 가벼운 로컬 중계 서버가 기본 동봉되어 있습니다.

1.  **필수 패키지 설치 (최초 1회):**
    ```bash
    npm install express socket.io chzzk
    ```
2.  **중계 서버 가동:**
    ```bash
    node chzzk_proxy_server.js
    ```
    *   *기본 포트(3000)가 다른 프로그램에 의해 사용 중인 경우*, 다음과 같이 실행 포트를 인자로 전달해 가동할 수 있습니다.
        ```bash
        node chzzk_proxy_server.js 3001
        ```

### 2. 실제 연동 방법
1.  **중계 서버 연결:** 게임 내 설정 패널의 `중계 서버 주소 (Socket.io)`에 가동한 서버 주소(예: `http://localhost:3000`)를 확인하거나 기입합니다.
2.  **치지직 주소 또는 고유 ID 기입:** 
    *   스트리머의 채널 전체 URL 주소(예: `https://chzzk.naver.com/live/bb324c...`)나 채널 고유 ID를 복사하여 `치지직 채널 ID 또는 방송 URL` 입력창에 붙여넣습니다. (전체 주소 붙여넣기 시 자동으로 고유 채널 ID를 추출하여 세팅합니다)
3.  **연동 시작:** **[연동 시작]** 버튼을 클릭하면 중계 소켓 서버에 조인(Join)되어 실시간 채팅이 동기화되며, UI 표시가 `연결됨 🟢`으로 변경됩니다.
4.  **연동 종료:** **[연동 종료]** 버튼을 누르면 리소스 누수 없이 로컬 Socket.io 커넥션이 정상 종료(Disconnect)되고 `연결 끊김 🔴`으로 복원됩니다.

## 🦖 인게임 주요 특징 및 물리 시스템

### 1. 적(몬스터) 겹침 방지 (Separation Force)
*   몬스터들이 플레이어를 추적할 때 한 점에 뭉쳐서 움직여 난이도가 급상승하고 시각적으로 겹쳐 보이던 문제를 개선했습니다.
*   물리 연산 루프 내에 **상호 척력(Separation Force)** 계산 로직이 내장되어 있습니다. 유닛 간의 상대 거리가 최소 기준(24px) 이내로 가까워지면 서로를 밀쳐내는 물리력이 보정 속도로 가산됩니다.
*   **지상 몬스터:** 지형 충돌 및 낙하 밸런스를 고려하여 X축 수평 방향으로만 반발력이 가해집니다.
*   **공중 몬스터(FLYER):** 사방으로 자연스럽게 퍼질 수 있도록 X/Y 전 방향으로 반발력이 적용됩니다.

### 2. 파워업 상자 디자인 테마 통일
*   인게임 던전에 생성되거나 보스를 잡고 나오는 아이템 파워업 상자의 표시 폰트가 투박하던 구형 monospace 9px에서, 미래지향적 분위기에 적합한 **`900 11px 'Orbitron', sans-serif`** 테마 폰트로 통일되었습니다.
*   상자의 바운딩 박스 `20x20px` 대비 정확한 가로/세로 중앙 정렬 및 렌더링을 지원하며, HP 복구 아이템(`SPEC_HP`)은 문자 `H`로 직관적이게 구분되어 출력됩니다.

### 3. 최대 낙하 속도 제한 (Terminal Velocity)
*   체공(제트팩 조작)이나 낙하 시 중력이 과도하게 계속해서 더해져 순식간에 수직 추락하는 조작감 저하 문제를 예방하기 위해 물리 엔진에 **종단 속도(Terminal Velocity, 10px/frame)** 개념을 탑재했습니다.
*   플레이어의 낙하 속도(`vy`), 적(몬스터)들의 낙하 속도, 드롭 아이템(골드/코인)의 낙하 속도 모두 프레임당 최대 10px 이상 가속되지 않도록 제한하여, 안정적이고 공평한 체공/플랫포밍 조작을 보장합니다.

### 4. 방패병(TANK)의 특수 방패 돌격 및 몬스터 접촉 대미지
*   **방패병(TANK) Shield Bash 공격 수단 구현:** 공격 패턴이 없던 방패병이 플레이어와 가까이(55px 이내) 근접하면 0.3초간 몸을 흔드는 선딜레이 경고 후, 전방으로 방패를 내밀어 돌진하는 공격을 감행합니다. 충돌한 플레이어는 **14 대미지**를 입고 뒤로 멀리 날아가는 **강력한 넉백 물리 효과**가 발생합니다.
*   **일반 적들의 몸싸움 접촉 대미지:** 기존에 닿아도 플레이어에게 대미지를 가하지 못하던 근접형 적들(박쥐 FLYER, 슬라임 SPLITTER) 및 대시 준비 중이 아닌 적들이 플레이어에게 닿을 때, 최소한의 몸싸움 접촉 피해(4~6)가 발생하도록 개선하여 던전 난이도 밸런스를 상향 개편했습니다. (대미지 적용 시 무적 프레임 1초 적용되어 중첩 피해 방지)

### 5. 초반 라운드 기본형 스폰 보장 (난이도 밸런싱)
*   초반 던전의 난이도가 불규칙하게 치솟는 것을 방지하기 위해, 화염/빙결/맹독/거대/은신 등의 접두사(Modifier)가 달린 변종 특수 적들은 **오직 4라운드부터 등장**하도록 설계했습니다.
*   1~3라운드에서는 접두사가 제거된 순수한 기본 형태의 적(궁수, 돌격병, 방패병 등)만 정직하게 출현하여 초반 조작 및 게임 메커니즘 적응을 돕습니다.

### 6. 플레이어 및 적 유도(Homing) 알고리즘 탑재 및 자석 범위 성장
*   **골드 자동 유도 (🧲 골드자석+ 스탯 신설):** 골드를 자동으로 흡수하는 자석 기능이 최초에는 닿기 직전 범위(30px)에서만 미세하게 작동하도록 하향 평준화하고, 상점의 새로운 카드 **"🧲 골드자석+"** 업그레이드를 통해 자석 적용 반경을 레벨당 50px씩 영구 확대(`30px + magnet * 50`px)되게 수정했습니다.
*   **탄환 적 유도 (🎯 유도탄+ 스탯 신설):** 상점 리스트에 무작위로 뜨는 새로운 특수 스탯 업그레이드인 **"🎯 유도탄+"** 카드가 신설되었습니다. 이 스탯이 1 이상일 때, 사격한 투사체가 300px 사거리 범위 내의 가장 가까운 적(은형 상태 적 제외)을 감지하여 각도를 선회 보정하며 타격하는 유도 투사체 궤적 계산이 가동됩니다. (업그레이드 스택당 선회 각도 속도가 향상됩니다)

### 7. 제한식 관통력 물리 도입 (🌀 관통력+ 스탯 신설)
*   투사체가 적을 무한히 관통하며 쓸어버리던 기존 메커니즘을 수정하여 **최대 관통 횟수(Pierce Limit)** 제한을 적용했습니다.
*   관통형 캐릭터(Seria)의 탄환은 최초 **1회 관통**을 지니며, 2번째 적에 충돌하면 즉시 소멸합니다.
*   상점의 신규 특수 카드 **"🌀 관통력+"** 업그레이드를 구매할 때마다 투사체의 적 관통 횟수가 1회씩 증가합니다. 관통 능력이 없는 캐릭터도 이 카드를 구매하면 즉시 적을 관통할 수 있는 능력을 획득합니다.

### 8. 지하 던전 맵 디자인 및 테마 개편
*   **지하 깊이에 따른 배경 그라데이션 변화:** 플레이어가 하단으로 하강하여 깊숙이 들어감(Round 상승)에 따라 갈색 돌벽(1~3F) -> 푸른 이끼 석벽(4~7F) -> 심연 보라/어둠(8~11F) -> 용암/마그마 층(12F~)으로 분위기가 실시간 전환됩니다.
*   **패럴랙스 배경 돌벽 및 부유 입자:** 기존의 우주 별 배경을 제거하고, 카메라 이동 속도에 대응하여 부드럽게 스크롤되는 **돌벽 균열(Parallax Stone Wall)** 무늬와 함께 공기 중에 몽환적으로 흩날리는 **지하 먼지/불씨 입자** 렌더링을 구현했습니다.
*   **던전 벽돌 발판 및 마법 룬 석판 도입:** 사이버 네온 스타일 발판을 폐기하고, 라운드 테마별 표면 디테일(이끼/마그마 띠 등)과 벽돌 줄눈 무늬가 반영된 **돌벽 블록(Solid Stone Bricks)**으로 교체했습니다. 또한 일방통행 발판은 은은하게 타오르는 **마법 룬 각인 석판(Floating Runic Slabs)** 테마로 새롭게 스킨했습니다.
*   **인게임 UI 판타지화:** 최상단 UI 타이틀 명칭을 사이버틱한 `STAGE R-X` 대신 `DUNGEON BxF` (Dungeon Depth) 포맷으로 수정하고, 네온 스타일 광원 그라데이션을 룬 광석 빛으로 매만져 던전 모험의 감성을 극대화했습니다.
